Java was created at *Sun Microsystems, Inc.*, where *James Gosling* led a team of researchers in an effort to create a new language that would allow consumer electronic devices to communicate with each other. It was released in 1995 as a core component of Sun Microsystems' Java platform.

### The naming convention
Firstly, it was called *"Greentalk"* by James Gosling, and the file extension was *.gt*.
After that, it was called *Oak* and was developed as a part of the Green project.
Why *Oak*? *Oak* is a symbol of strength and chosen as a national tree of many countries like the U.S.A., France, Germany, Romania, etc.

In 1995, *Oak* was renamed as *"Java"* because it was already a trademark by Oak Technologies.
Java is an island of Indonesia where the first coffee was produced (called java coffee). It is a kind of espresso bean. Java name was chosen by James Gosling while having coffee near his office.

### Architecture
The Java Compiler converts the java code into byte codes. The Java Virtual Machine (JVM) converts the byte codes into machine code which the machine executes directly. 

We can describe this process in the following steps:
1. The source code goes to the Java Compiler
2. The Java Compiler converts it into byte codes
3. The byte codes go to the Java Virtual Machine 
4. Finally, it goes to the OS (operating system)
